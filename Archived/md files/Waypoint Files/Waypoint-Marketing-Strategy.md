## Website Strategy

### Do You Need a Website? Yes, But Simple.

**Purpose:**
1. App Store link hub
2. Demo video hosting
3. Press kit / media assets
4. Email capture (future)

**Don't Need:**
- Stripe integration (App Store handles payments)
- User accounts / auth
- Database (Supabase not needed)

‚úÖ **Vercel:**
- Free hosting

- Fast global CDN
- Zero config

‚úÖ **Tailwind CSS:**
- Utility-first (easy to style)
- Looks modern by default
- Small bundle size

- Pre-built components (buttons, cards, etc.)

- Looks professional

**What You DON'T Need:**
- ‚ùå Stripe (App Store handles billing)

- ‚ùå Auth (no login needed)
